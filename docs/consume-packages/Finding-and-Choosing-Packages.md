## <a name="search-syntax"></a><span data-ttu-id="487df-181">Arama söz dizimi</span><span class="sxs-lookup"><span data-stu-id="487df-181">Search Syntax</span></span>

<span data-ttu-id="487df-182">NuGet paket arama aynı nuget.org, NuGet CLI ve Visual Studio'da NuGet paket yöneticisini uzantısı içinde çalışır.</span><span class="sxs-lookup"><span data-stu-id="487df-182">NuGet package search works the same on nuget.org, from the NuGet CLI, and within the NuGet Package Manager extension in Visual Studio.</span></span> <span data-ttu-id="487df-183">Genel olarak, arama paket açıklamaları yanı sıra anahtar sözcükleri uygulanır.</span><span class="sxs-lookup"><span data-stu-id="487df-183">In general, search is applied to keywords as well as package descriptions.</span></span>

- <span data-ttu-id="487df-184">**Anahtar sözcükler**: arama için sağlanan anahtar sözcükleri içeremez ilgili paketleri arar.</span><span class="sxs-lookup"><span data-stu-id="487df-184">**Keywords**: Search looks for relevant packages that contain any of the provided keywords.</span></span> <span data-ttu-id="487df-185">Örnek: `modern UI`.</span><span class="sxs-lookup"><span data-stu-id="487df-185">Example: `modern UI`.</span></span> <span data-ttu-id="487df-186">Bir hüküm arasında paketlerin tüm sağlanan anahtar sözcükleri içeren, kullanmak için "+" gibi aranacak `modern+UI`.</span><span class="sxs-lookup"><span data-stu-id="487df-186">To search for packages that contain all of the provided keywords, use "+" between the terms, such as `modern+UI`.</span></span>
- <span data-ttu-id="487df-187">**İfadeleri**: Bu koşulları tam duyarlı eşleşme yapar terimleri tırnak içine girme.</span><span class="sxs-lookup"><span data-stu-id="487df-187">**Phrases**: Entering terms within quotation marks looks for exact case-insensitive matches to those terms.</span></span> <span data-ttu-id="487df-188">Örnek: `"modern UI" package`</span><span class="sxs-lookup"><span data-stu-id="487df-188">Example: `"modern UI" package`</span></span>
- <span data-ttu-id="487df-189">**Filtreleme**: sözdizimini kullanarak, belirli bir özellik için bir arama terimi uygulayabilirsiniz `<property>:<term>` burada `<property>` (büyük-küçük harf duyarsız) olabilir `id`, `packageid`, `version`, `title`, `tags`, `author`, `description`, `summary`, ve `owner`.</span><span class="sxs-lookup"><span data-stu-id="487df-189">**Filtering**: You can apply a search term to a specific property by using the syntax `<property>:<term>` where `<property>` (case-insensitive) can be `id`, `packageid`, `version`, `title`, `tags`, `author`, `description`, `summary`, and `owner`.</span></span> <span data-ttu-id="487df-190">Koşulları gerekirse tırnak içinde yer alabilir ve aynı anda birden çok özellik için arama yapabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="487df-190">Terms can be contained in quotes if needed, and you can search for multiple properties at the same time.</span></span> <span data-ttu-id="487df-191">Ayrıca, arama yapar `id` özelliği olan eşleşmelerini, oysa `packageid` tam bir eşleşme kullanır.</span><span class="sxs-lookup"><span data-stu-id="487df-191">Also, searches on the `id` property are substring matches, whereas `packageid` uses an exact match.</span></span> <span data-ttu-id="487df-192">Örnekler:</span><span class="sxs-lookup"><span data-stu-id="487df-192">Examples:</span></span>

    ```
    id:NuGet.Core                # Match any part of the id property
    Id:"Nuget.Core"
    ID:jQuery
    title:jquery                 # Searches title as shown on the package listing
    PackageId:jquery             # Match the package id exactly
    id:jquery id:ui              # Search for multiple terms in the id
    id:jquery tags:validation    # Search multiple properties
    id:"jquery.ui"               # Phrase search
    invalid:jquery ui            # Unsupported properties are ignored, so this
                                 # is the same as searching on jquery ui
    ```
